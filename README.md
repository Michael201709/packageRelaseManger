# sflog
软件发布记录。可快速管理软件。无需数据库支持。纯markdown

配置允许访问的IP。

	ipallow.php

设置软件包
	
	markdown                    --软件包列表
		shop                    --软件名
			readme.txt          --软件说明
			licence             --版本号
				tip.txt         --列表页，简要更新提示
				readme.txt      --详情页，更新的主要内容。
				licence.zip     --与版本号相同的zip包，供下载使用。
